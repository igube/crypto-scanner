# Crypto Trading Alert System

This is my Python project where I built a simple system for monitoring the cryptocurrency market and generating alerts based on specific conditions.

I created this project to better understand automation, working with market data, and building simple systems that can check data automatically and notify the user when something important happens.

## Why I built this project

The cryptocurrency market changes very quickly, and manually checking prices or market signals can be time-consuming. I wanted to build a simple system that could monitor market data automatically and generate an alert when a selected condition is met.

I treat this project mainly as a learning project focused on automation, alert logic and Python.

## What I built

In this project, I created a system that:

- monitors market data,
- checks selected conditions,
- generates alerts,
- automates the process of market monitoring,
- can be expanded with more alert rules in the future.

This is not a finished investment product. It is an educational and technical project.

## Technologies

In this project I used / planned to use:

- Python
- basic automation logic

- alert conditions

## How it works

The general logic of the project is:

1. The system collects or receives market data.
2. The program checks if the data meets selected conditions.
3. If a condition is met, the system generates an alert.
4. The alert can be displayed or sent to the user.
5. The process can be repeated automatically.
